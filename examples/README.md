# Contoh Tenun

Jalankan: `tenun run examples/<nama>.tenun`

## Dasar
- `hello.tenun` тАФ Halo Dunia
- `faktorial.tenun` тАФ rekursi
- `fibonacci.tenun` тАФ rekursif + iteratif
- `fizzbuzz.tenun` тАФ kontrol alur
- `larik.tenun` тАФ larik dasar

## Fitur bahasa
- `peta_json.tenun` тАФ peta dinamis / objek JSON bersarang
- `fungsi_nilai.tenun` тАФ first-class function + higher-order
- `galat.tenun` тАФ coba/tangkap (error handling)
- `cocok.tenun` тАФ pencocokan nilai (switch)
- `bitwise.tenun` тАФ operator bitwise + flag + hex/biner

## Algoritma & utilitas
- `urut.tenun` тАФ bubble sort
- `statistik.tenun` тАФ sum/rata/maks/min (for-each)
- `teks.tenun` тАФ operasi string (pisah/gabung/pangkas/keBesar/ganti)
- `kalkulator.tenun` тАФ cocok pada operator
- `waktu_acak.tenun` тАФ waktu(), acakAngka(), token acak
- `bench.tenun` тАФ benchmark loop

## Web (server bawaan)
- `server.tenun` тАФ server HTTP via `layani` + `tangani`
- `web.tenun`, `webapp.tenun` тАФ app web + file statis (folder `publik/`)

Contoh penggunaan modul (web framework, ORM, dll) ada di tiap repo modul:
`github.com/TenunLang/modul-<nama>/contoh/`, dan aplikasi penuh di
`github.com/TenunLang/toko-tenun`.
